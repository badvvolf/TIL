# Same Origin Policy (SOP)

## 개념
* 다른 출처에서 온 리소스를 격리해 공격 경로를 줄인다. 
* 동일 프로토콜/호스트/포트 이면 `scheme/host/port 튜플` 또는 `튜플` 이라 부른다. 
* [CORS](/document/Network/HTTP/cross-origin resource sharinng.md)를 이용하여 다른 출처를 허용할 수 있다.
